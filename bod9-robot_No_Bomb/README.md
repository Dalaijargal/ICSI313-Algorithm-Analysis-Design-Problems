input: NxN array орж ирэх ба хад байвал тухайн нүдний утга -1 зоос байвал +1 энгийн нүд байвал 0.

| 0 | -1 | 0 | 0 | R |
| --- | --- | --- | --- | --- |
| 0 | 0 | 1 | -1 | 0 |
| 1 | 1 | 0 | -1 | -1 |
| 0 | 0 | 0 | 1 | 1 |
| F | 0 | 0 | -1 | 1 |


```python
for i in array:
	for j in i:
		if(dp[i][j]!=-1):
			DP[i][j]=max(DP[i+1][j],DP[i][j+1])+DP[i][j]
		else:
			DP[i][j]=-1
```

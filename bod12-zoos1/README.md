# Шалгалт

Чамд C1 C2 C3 гэсэн зоосний хэмжээ болон n бүхэл тоо өгөгдөв тухайн төрлийн зоосноос хангалттай олон байгаа.

Хамгийн бага зоос ашиглан хэрхэн задлах вэ, задлаж чадахгүй бол -1 буцаана.


```python
dp[i] = min(dp[i-c1],
dp[i-c2],
dp[i-c3]
)+1
```
